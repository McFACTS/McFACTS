McFACTS Style Guide
=========================

See the `Contributing Guide <https://github.com/McFACTS/McFACTS/blob/main/docs/source/contribute.rst>`_ for more
information on making changes to McFACTS.


Please follow these guidelines as you work on McFACTS.

Documentation
*************

We try to follow `Numpy standard <https://numpydoc.readthedocs.io/en/latest/format.html>`_ for class, method,
and function documentation
